Merupakan aplikasi pengelolaan rumah sakit
Aplikasi ini ditujukan untuk admin dari sebuah rumah sakit.
Adapun fungsionalitasnya antara lain:
- Input
	- data dokter
	- data pasien
	- data ruangan
	- data pasien inap
	- data diagnosa
- Hapus
	- data dokter
	- data pasien
	- data ruangan
	- data pasien inap
	- data diagnosa
- Cari
	- data dokter
	- data pasien
	- data ruangan
- Lihat
	- data dokter
	- data pasien
	- data ruangan
